## 2. 결과물 현황 (`index.html`)

- **형식**: 단일 HTML 파일. CSS는 `<style>` 인라인, JS는 하단 `<script>` 인라인.
- **이미지**: 사업계획서(PPTX)에서 추출한 사진 3장을 **base64로 내장**(외부 파일 의존성 없음).
  - 히어로 배경 = 새벽 들녘(새벽배송 컨셉)
  - 산지 직거래 띠 배경 = 싱싱한 채소잎
  - 사회적 가치 섹션 배경 = 초록잎
- **폰트**: 본문 Pretendard(jsDelivr CDN), 디스플레이 Gowun Batang(Google Fonts) — 인터넷 연결 필요.
- **배포**: 파일 하나라 Netlify Drop / Vercel / 일반 웹호스팅에 그대로 올리면 됨.

### 디자인 토큰 (CSS `:root` 변수)
- `--pine:#16352A` (메인 딥그린, 60~70% 비중)
- `--leaf:#6FA838` / `--leaf-bright:#8FC74A` (포인트 그린)
- `--gold:#C8912E` (인증·강조 소량)
- `--paper:#FBFCFA` / `--mint:#EAF2E2` (밝은 배경)
- `--ink:#192219` (본문 텍스트) / `--muted:#5E6B62`
- 모티프: 잎사귀 SVG 글리프(섹션 eyebrow 앞 아이콘으로 반복). **밑줄·색띠(accent bar) 사용 금지**(AI 티 남).

### 섹션 구성 순서 (id)
1. `#top` 헤더(고정 내비)
2. `#top` 히어로 — "아이들의 급식, 27년 한 길로 지킵니다"
3. `#greet` 인사말(대표 인용 카드 + 회사 핵심정보 facts)
4. `#school` **학교급식 전문**(강점 6카드 + 지표 4개) ← 핵심 섹션
5. `#safety` **안전·품질**(검품 3단계 + 부적합품 차단 + 안전성 검사)
6. `#history` 연혁(1997→2024 타임라인)
7. 산지 직거래 띠(채소 사진 배경)
8. `#certs` 인증(6카드)
9. `#cold` 콜드체인(새벽 배송 스케줄 타임라인 + 위생 3카드)
10. `#clients` 거래처(급식 / 외식)
11. 사회적 가치(잎 배경)
12. `#contact` 오시는 길·문의
13. 푸터

### 동작
- 헤더 스크롤 시 `.scrolled` 토글(투명→반투명 흰색).
- `.reveal` 요소 IntersectionObserver로 스크롤 등장.
- 모바일(≤880px) 햄버거 메뉴, 그리드 1열, 타임라인 세로 변환.
- `prefers-reduced-motion` 대응.

- 문구는 마케팅 톤이되 과장·허위 금지(행정처분 이력 없음 등 사실 기반 유지).
